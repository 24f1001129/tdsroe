DevOps Challenge: Matrix Build

This repository demonstrates a GitHub Actions matrix build.

The workflow in .github/workflows/matrix-build.yml is configured to:

Run a matrix build for 3 different Python versions (3.9, 3.10, 3.11).

Generate a unique artifact for each build.

Upload each artifact with the required prefix build-bfedd75-.
